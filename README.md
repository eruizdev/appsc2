# VelkyVet

App de gestión veterinaria para Android. Proyecto base.

## Stack

- Kotlin
- Jetpack Compose
- Room Database
- Navigation Compose
- Material Design 3

## Requisitos

- Android Studio Ladybug o más reciente
- JDK 17
- Android 8.0+ (API 26)

## Cómo correrlo

1. Clonar el repositorio
   ```bash
   git clone [URL_DEL_REPOSITORIO]
   ```

2. Abrir la carpeta en Android Studio con **File → Open**

3. Esperar que Gradle sincronice

4. Dale **Run ▶️**

## Ramas

- `main` - versión estable
- `feature/auth` - pantallas de login, registro y recuperar contraseña
- `feature/home` - pantalla principal y navegación
- `feature/mascotas` - gestión de mascotas con Room
- `feature/citas` - agendar y listar citas
- `feature/reportes` - pantalla de reportes
- `feature/perfil` - perfil del usuario
